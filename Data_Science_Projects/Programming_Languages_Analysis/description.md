# **Programming Languages Trends Analysis (2008–2020)**

### **Overview**  
Analyzed Stack Overflow post data (2008–2020) to identify programming language popularity trends and translate them into strategic insights for businesses, developers, and educators.

### **Business Problem**  
Provide data-driven insights for:  
- Tech stack and investment decisions  
- Hiring and skill planning  
- Curriculum and career development  

### **Dataset**  
- **Source:** Stack Overflow monthly post counts  
- **Period:** Jul 2008 – Jul 2020 (145 months)  
- **Languages:** 14 (Java, Python, JS, C#, C++, Go, R, etc.)  
- **Records:** 1,991 monthly observations  

### **Implementation**  
- Cleaned and formatted raw CSV data  
- Conducted EDA and time series analysis  
- Analyzed correlations, growth, and volatility  
- Visualized trends and created market insights  

### **Technologies**  
Python, Pandas, NumPy, Matplotlib, Jupyter  

### **Key Findings**  
- **Leaders:** JavaScript (web), Python (data/ML), Java (enterprise)  
- **High Growth:** Python, JS, Go  
- **Stable:** Java, C#  
- Clear seasonal peaks and correlated tech trends  

### **Impact**  
- **Companies:** Optimize hiring & investments  
- **Developers:** Prioritize in-demand skills  
- **Educators:** Align programs with industry needs  

### **Skills Shown**  
Data wrangling, time series analysis, visualization, statistical analysis, and strategic insight generation.
